La sesión de trabajo ha terminado. Actualiza el fichero `DEVLOG.md` añadiendo una nueva entrada al final con este formato exacto:

```
### HH:MM — [Título breve de lo hecho]
- Descripción detallada de cada cosa realizada
- Ficheros creados: lista con ruta relativa y descripción
- Ficheros modificados: lista con ruta relativa y qué cambió
```

También actualiza el ROADMAP.md marcando como `[x]` cualquier tarea que se haya completado durante esta sesión.

Al final del DEVLOG añade siempre una sección:
```
## Pendiente para próxima sesión
- [ ] Tarea 1
- [ ] Tarea 2
...
```

Usa la fecha y hora actuales del sistema para los timestamps.
